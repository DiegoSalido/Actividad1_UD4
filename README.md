# Actividad 1 - Refactorización en Java

## Descripción
Programa en Java que analiza un array de números.
Calcula el número máximo, comprueba si se repite y calcula la media.

## Cómo ejecutar el programa
1. Compilar:
   javac NumberAnalyzer.java
2. Ejecutar:
   java NumberAnalyzer
